# TUUO Asset Inventory - Project Summary & LinkedIn Post

---

## 🔧 Tech Stack

| Layer | Technology |
|-------|------------|
| **Frontend** | React 18, TypeScript, Tailwind CSS, Vite |
| **UI Components** | Radix UI, Lucide React, Recharts |
| **State Management** | React Query, React Router DOM |
| **Backend** | Express.js, Node.js, MongoDB + Mongoose |
| **Auth** | JWT (JSON Web Token) |
| **Data Import** | Multer, PapaParse (CSV) |

---

## 📋 Pages & Features

| Page | Purpose |
|------|---------|
| Login | Secure JWT authentication |
| Dashboard | 5 KPI cards, Asset Status, Department & Floor distribution |
| Assets | Full list with filters, search, pagination |
| Add Asset | Create new asset with 40+ fields |
| Asset Detail | View/Edit asset, Assign/Unassign |
| Maintenance | Schedule & track maintenance logs |
| Parts/Components | Track component changes (RAM, HDD, PSU, etc.) |
| Stock | Stock management by department |
| Store Inventory | Store-level stock tracking |
| Departments | View assets grouped by department |
| Floors | View assets grouped by floor/location |
| Reports | Export CSV/JSON |
| Import | Bulk import via CSV |
| Live Map | PRTG network map integration |
| Settings | User profile & app settings |

---

## 🗃️ Data Model

- **Assets**: 40+ fields (SL, Type, Hostname, IP, Status, Department, Floor, Hardware specs, Warranty, etc.)
- **Stock**: Item type, quantity, department, location
- **Maintenance Logs**: Asset SL, type, due date, status
- **Parts Changes**: Old→New components, cost, date, reason
- **Users**: Auth & authorization
